## AMIとは

**AMI**（アミ）は、足の８０２０ウォーキングが定める独自の社会貢献参考単位です。

- 「歩み」（日本語）＋「友人」（仏語 Ami）＋「仲間」（西語 Amigo）
- 1 AMI = 1円換算の参考値
- **AMIはあくまで個人の目安・動機づけのための参考値です**
- 実際の社会保障費削減を保証・約束するものではありません
- 寄付や社会保障制度への直接的な還元を意味するものではありません
- 健康効果を医学的に保証するものではありません

---

## できること

- **5月イベント参加**の記録をAMIとして入金
- **通年歩行記録**（月次）をAMIとして積み立て
- 累計AMI残高・今年の歩行達成率をリアルタイム表示
- 年次サマリー通帳を印刷・PDF保存
- X（旧Twitter）へのSNSシェア
- 「このアプリを保存」ボタンでHTMLファイルをダウンロード
